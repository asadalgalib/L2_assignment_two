
# Rent A Ride

Rent a Ride is a backend rental system with JWT-based authentication for secure user access. Users can book vehicles, manage their data, cancel bookings before start dates, while admins are restricted from deleting accounts with active bookings.

## Run Locally

Clone the project

```bash
  https://github.com/asadalgalib/L2_assignment_two.git
```

Go to the project directory

```bash
  cd L2_assignment_two
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```


## Environment Variables

To run this project, you will need to add the following environment variables to your .env file

Database connection - `CONNECTION_STR` 

Json Web Token secrete - `JWT_SECRET`
